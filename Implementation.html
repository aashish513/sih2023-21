Working

1. Data Acquisition: Cameras capture real-time images. Data sent to Raspberry Pi for initial processing. 2. Person Detection: AI Model analyzes images to identify human beings. Results communicated to the control system. 3. Alarm Generation: Automatic alarm triggered upon person detection. Notification sent to the rescue team. 4. Payload Dropping: Control system calculates drop location based on person's position. Payload is released with precision.

Setup AND Calibration

  1. Drone Configuration: Ensure stable flight, GPS accuracy, and payload capacity.
  2. Camera Installation: Mount cameras for optimal coverage.
  3. Processor Board Setup: Connect and configure Raspberry Pi / Intel NUC.
  4. AI Model Integration: Install and configure the neural network.
  5. Payload Mechanism Calibration: Ensure accurate payload delivery.
